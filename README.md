# WeCheat

基于 MonkeyDev 对 WeChat 的个人扩展。

[![](https://img.shields.io/badge/MonkeyDev-v1.0-lightgrey.svg)](http://monkeydev.org/)
[![](https://img.shields.io/badge/WeChat.ipa-v6.5.15-green.svg)]()

## Features

- [ ] 中心控制按钮，用于开启以下功能
- [x] 消息防撤回
- [x] 微信运动每日刷步数
- [x] 夜间模式
- [ ] 交互猜拳作弊功能
- [ ] 自动抢红包功能
